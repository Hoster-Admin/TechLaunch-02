# 📐 Prototypes

These are the original self-contained HTML prototypes built before the full-stack React/Node.js app.
They run completely standalone — just open in a browser, no server needed.

| File | Description | Size |
|------|-------------|------|
| `tech-launch-v2.html` | Full user-facing site — home feed, product pages, user profiles, entities (accelerators/investors/studios), DMs, notifications, submit flow, bookmarks, persona system | ~369KB |
| `tech-launch-admin.html` | Admin panel v9 — Dashboard, Products (approve/reject/feature), Users, Entities, Applications, Featured, Reports, Settings, My Profile. Includes all 26 bug fixes from audit. | ~140KB |

## How to use

Just open either file directly in your browser:

```bash
open prototypes/tech-launch-v2.html
open prototypes/tech-launch-admin.html
```

## Admin panel demo credentials (prototype)
- These are hardcoded in the HTML — no login required, just open and use.

## Relationship to full-stack app

The prototypes were the **design & UX reference** for building the production app in `/backend` + `/frontend`.
All features, flows, data structures and design decisions in the React app are based on these prototypes.
